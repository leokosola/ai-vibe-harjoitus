Tunnista valtio mille lippu kuuluu ja kirjoita se tekstikenttään. Voit myös ottaa vihjeen, joita on viisi per maa. Vihjeet ovat tekoälyn kaivamia "faktoja" kyseisestä maasta (voivat olla totta sitten eivät).

Arvuuteltavat maat ovat tällä hetkellä kaikki Euroopan maita. Tulevaisuudessa peliä voisi laajentaa kaikkiin maailman maihin.
