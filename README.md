Tunnista valtio mille lippu kuuluu ja kirjoita se tekstikenttään. Voit myös ottaa vihjeen, joita on viisi per maa. Vihjeet ovat tekoälyn kaivamia "faktoja" kyseisestä maasta (voivat olla totta tai sitten eivät, pikatarkistuksen perusteella totta).

Arvuuteltavat maat ovat tällä hetkellä kaikki eurooppalaisia. Tulevaisuudessa peliä voisi laajentaa kaikkiin maailman maihin, jotta pelissä olisi aidosti haastetta.

Pelaa: [leokosola.github.io/ai-vibe-harjoitus](https://leokosola.github.io/ai-vibe-harjoitus/)
